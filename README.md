


# Next Level Week 4

Este projeto é uma API capaz de enviar pesquisas de satisfação via email para cálculo do NPS da empresa.
Desenvolvido durante a Next Level Week 4 da Rocketseat.

## Começando

As instruções a seguir irão te permitir rodar o sistema em modo de desenvolvimento.

### Pré-requisitos

- [Node.js](https://nodejs.org/en/)

### Installation

Clone o repositório:
```
https://github.com/IsacJr/Next-Level-Week-4.git
```

#### Backend

Instale as bibliotecas, execute as migrations do banco e rode o projeto com os comandos abaixo:
 ```
cd Next-Level-Week-4/node
npm install
npm run typeorm migration:run
npm run dev
```
